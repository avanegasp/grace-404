# grace-404
Grace 404 — IA + Biblia como fuente de esperanza. Una micro-app construida en 30 minutos una noche con HTML, JS y Groq API. Porque a veces buscas y encuentras.

Creada con HTML, CSS, JavaScript y una API key de Groq conectada al modelo LLaMA 3.3 —gratuito, rápido y accesible. Le di un "prompt" específico: "Responde con un tono amoroso, esperanzador y un versículo bíblico al final de cada respuesta".

Grace 404 incluye las siguientes caracterísiticas:

- Conectarse a un API KEY de Groq
- Puedes colocar tu foto 
- Elegir en qué género prefieres que te responda. 
- Compartir el link para comenzar a usarla por Whatsapp 
- Cambiar de idioma En / Es
- Reiniciar las respuestas
